# Wingsuit Kickstart
Wingsuit Kickstart is a demo project to show how to use Wingsuit with Drupal.

## Installation instruction

   * Clone the repository
   * run composer install
```
    composer install
```
   * Install drupal with blt
```
    ./vendor/bin/blt stup
```
